# CampusFlow — Multi-User Cloud Edition

CampusFlow now supports separate student accounts through Supabase Auth.

Each student receives:
- A private profile
- A private task list
- Private automation logs
- Their own WhatsApp phone number
- Their own email address

Read `MULTIUSER_SETUP.md` before redeploying.

Important: the Supabase service role key belongs only in Render.
