# CampusFlow

CampusFlow is configured for **cloud deployment**, so the final application can run through a public website instead of localhost.

Recommended deployment:

- Frontend on Vercel
- Backend on Render
- Workflows on n8n Cloud
- WhatsApp through Twilio Sandbox
- Google Calendar through n8n OAuth

See `DEPLOYMENT.md` for the full online deployment procedure.

## Tech stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Storage: Local JSON file
- AI: Groq API, with an offline fallback summarizer
- Automation: n8n webhooks
- WhatsApp: Twilio node inside n8n
- Calendar: Google Calendar node inside n8n

## 1. Install

From the project root:

```bash
npm install
npm run install:all
```

## 2. Configure backend

Copy:

```bash
backend/.env.example
```

to:

```bash
backend/.env
```

Then update the values.

Minimal setup:

```env
PORT=5000
FRONTEND_URL=http://localhost:5173
```

Optional AI integration:

```env
GROQ_API_KEY=your_groq_api_key
GROQ_MODEL=llama-3.1-8b-instant
```

Optional n8n integration:

```env
N8N_DEADLINE_WEBHOOK_URL=https://your-n8n-domain/webhook/deadline
N8N_NOTICE_WEBHOOK_URL=https://your-n8n-domain/webhook/notice
N8N_ATTENDANCE_WEBHOOK_URL=https://your-n8n-domain/webhook/attendance
```

The app still works without Groq or n8n. It will use a local summarizer and log automation requests as skipped.

## 3. Run

From the project root:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

Backend runs at:

```text
http://localhost:5000
```

## 4. Demo flow

1. Complete student onboarding.
2. Add a deadline.
3. Open the Automations page and verify the webhook status.
4. Paste a college notice and summarize it.
5. Broadcast the notice to n8n.
6. Calculate attendance risk.
7. Show the Google Calendar event and WhatsApp message generated by n8n.

## 5. n8n workflow setup

Import the JSON files inside `n8n-workflows`.

### Deadline workflow

Use these nodes:

```text
Webhook
  → Set/Edit Fields
  → Google Calendar
  → Twilio WhatsApp
  → Respond to Webhook
```

For a live hackathon demo, send the WhatsApp confirmation immediately. A Wait node can be added later for 24-hour and 1-hour reminders.

### Notice workflow

Use these nodes:

```text
Webhook
  → Set/Edit Fields
  → Google Calendar
  → Twilio WhatsApp
  → Respond to Webhook
```

### Important

After importing:

- Connect your Google Calendar credential.
- Connect your Twilio credential.
- Replace placeholder WhatsApp sender numbers.
- Activate the workflows.
- Copy production webhook URLs to `backend/.env`.

## 6. Twilio WhatsApp Sandbox

1. Open Twilio Console.
2. Go to Messaging → Try it out → WhatsApp.
3. Join the sandbox from your phone.
4. Set the Twilio credential in n8n.
5. Use phone numbers in international format, for example:

```text
+919876543210
```
